## Development

#### Clone this repo

```bash
$ git clone https://github.com/republicprotocol/swapperd-native
$ cd swapperd-native
```

#### Install dependencies

```bash
$ yarn
```

#### Launch dev environment

```bash
$ yarn run dev
```

#### Package application

This will package it for the same OS it's being run on.

```bash
$ yarn run package
```
